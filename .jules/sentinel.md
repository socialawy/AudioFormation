## 2024-05-24 - [Path Traversal Bypass via Symlinks]
**Vulnerability:** The `validate_path_within` function in `src/audioformation/utils/security.py` used string manipulation (`os.path.abspath`) to verify that a path resided within a root directory. However, string manipulation does not follow symlinks. This meant that a path could be a symlink pointing outside the root directory, yet still pass validation because its string representation appeared to be inside the root directory.
**Learning:** Checking whether a path is within a root directory by using string comparisons and `os.path.abspath` fails to properly prevent path traversal bypasses. It is vulnerable to symlink attacks, where the string representation of a path differs from its resolved location on the disk.
**Prevention:** Path validation logic must always use `Path.resolve().is_relative_to(resolved_root)` to securely handle and resolve symlinks. Also catch exceptions like `(TypeError, ValueError, RuntimeError)` when performing the resolution to gracefully handle malformed paths and infinite symlink loops.

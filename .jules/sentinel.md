## 2024-05-18 - [Path Traversal bypass via Symlink resolution]
**Vulnerability:** The `validate_path_within` function in `src/audioformation/utils/security.py` previously used `os.path.abspath` to ensure a target path is located within a designated root directory. However, `os.path.abspath` solely relies on string manipulation and does not resolve symlinks. An attacker could craft a symlink within the root directory that points to an arbitrary file or directory outside the root directory, successfully bypassing the validation check and achieving directory traversal.
**Learning:** Pure string manipulation (like `os.path.abspath`) is insufficient for secure path validation when symlinks can be introduced by users or external processes, as it fails to evaluate the canonical destination of the requested path.
**Prevention:** Always use `pathlib.Path.resolve()` to normalize paths and resolve symlinks before validating path constraints. Ensure validation operates on canonical, resolved filesystem paths using methods such as `Path.is_relative_to()`.
